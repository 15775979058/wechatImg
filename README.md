# wechatImg
手机摄影大赛-网站应用

# 功能 #
- 照片上传
- 列表显示作品
- 投票、分享
- 作品修改

# 设及知识点 #
1. MVC开发模式
2. PHP+MySQL
3. 上传前，在前端利用JS对图片进行压缩（/Public/js/localCompress.js）
4. 微信公众平台-微信网页授权（获取用户openid、昵称、头像...）
5. 微信公众平台-微信JS-SDK(定制detail分享内容:标题、副标题、链接、图标...)
6. 利用微信openid限定一人投一票
